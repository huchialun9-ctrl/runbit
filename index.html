Core Technologies

Virtual File System

Files stored in IndexedDB with full path mapping. Supports drag-drop import and real-time read/write operations.

Service Worker Proxy

Intercepts all preview iframe requests. Serves files from VFS with correct MIME types. No network calls needed.

File System Access API

Direct two-way sync with your physical disk. Edit in VS Code, see changes instantly in the browser preview.

ES Modules Native

Import Maps + Service Worker enables native ES Module execution without bundlers. Modern JS just works.

100% Offline Capable

Everything runs in your browser. Your code never leaves your machine. No cloud, no uploads, no telemetry.

Iframe Sandbox Protection

Preview runs in a strictly sandboxed iframe. Scripts cannot escape to access the editor, cookies, or local storage.

Same-Origin Isolation

Virtual domain isolation ensures preview code cannot read or modify the host application's data or database.

No Server Dependencies

Zero backend, zero dependencies. Safe for corporate environments with strict network policies.

No Sign-ups Required

No accounts, no registration, no data collection. Open the page and start working immediately.

PWA Offline Support

Install as a Progressive Web App. Works offline after first load. Your dev tool is always available.
